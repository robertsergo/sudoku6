Description:
Ce site permet de jouer un jeu de sodoku 




Utilisation:
    Règlements et mode d'utilisation du jeu
        En ouvrant la page du jeu sodoku, vous serez en présenece
        d'un interface de jeu qui contient des lignes et des colonnes
        Il y a une ligne au-dessous de l'interface du jeu qui contient les
        chiffres de 1 à 9. Vous choisissez un chiffres pour le mettre dans la
        bonne case. Cliquer sur le chiffre désigné pour le selectionner et
        le placer dans une case que vous voulez.


        Les Règlements du jeu sont:

        Chaque ligne doit avoir un chiffre unique de 1 à 9.
        Chaque colonne doit avoir un chiffre unique de 1 à 9.
        Si vous placer un chiffre dans une mauvaise case,
        Le jeu vous affiche le nombre de fautes commises
        Vous gagnez le jeu quand toutes les cases, sont
        remplies et le compteur reste à zéro.
        Si vous placer un chiffre dans une bonne case,
        le compteur des fautes ne bouge pas.
